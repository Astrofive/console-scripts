# How to use

1. Click on the script you'd like to run above (ex **"button-clicker"**) and copy the code.
2. Right click on the webpage you'd like to run the script on.
3. Click "Inspect"
4. Click on the "Console" tab
5. Paste the code (you may have to type "allow pasting" to use the console for the first time)
6. Hit ENTER to run the script.
